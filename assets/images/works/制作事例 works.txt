制作事例 works
【参考サイト：https://giginc.co.jp/works】


01. 【建築・不動産】
クライアント名： ARCHI-BASE（建築設計事務所）

カテゴリタグ： コーポレートサイト UI/UXデザイン ブランドサイト
Corporate Website / Branding

（企業のブランディング、世界観を伝えるWebサイト）


実績タイトル： 建築の美しさを五感で伝える、余白を活かしたポートフォリオサイト構築

プロジェクト概要： 「作品である建築そのものを主役にしたい」という要望に応え、装飾を極限まで削ぎ落とし、大判の写真が引き立つグリッドレイアウトを実装。



02. 【美容・コスメ】
クライアント名： soi（オーガニックスキンケアブランド）

カテゴリタグ： ECサイト ブランディング HTML5/CSS3
EC Website / UI / UX Design

実績タイトル： ブランドの思想を1pxの繊細なラインで表現したECサイトリニューアル

プロジェクト概要： 20〜30代をターゲットに、アースカラーと洗練されたタイポグラフィで世界観を統一。商品のこだわりを伝えるため、各所にモーダルでの詳細表示を採用。


03. 【飲食・空間】
クライアント名： CROSS CAFE（コワーキングカフェ）

カテゴリタグ： 店舗サイト レスポンシブ対応 JavaScript
Brand & Promotion / Landing Page

実績タイトル： 赤坂の街に馴染む、機能性とデザイン性を両立した店舗案内サイト

プロジェクト概要： 利用システムや料金プランの複雑さを解消するため、情報設計（UI/UX）を徹底的に見直し。ユーザーが迷わないよう、料金表にはアコーディオンを導入。


04. 【IT・SaaS】
クライアント名： NextCore inc.（データ分析スタートアップ）

カテゴリタグ： コーポレートサイト 情報設計 Figma
Corporate Website / Information Architecture

実績タイトル： 抽象的なテクノロジーを直感的に可視化する、新世代の企業サイト制作

プロジェクト概要： 複雑なサービス内容を整理し、文字の強弱（ジャンプ率）と直線のコンポーネントを活かして構築。Figmaでのワイヤーフレームの段階から構造美を追求。



05. 【伝統工芸・小売】
クライアント名： うるし 門田（老舗漆器店）

カテゴリタグ： ブランドサイト 写真メイン フロントエンド
EC Website / Brand Storytelling

実績タイトル： 職人の手仕事を現代に紡ぐ、トレンドに流されない普遍的なWebデザイン

プロジェクト概要： 歴史あるブランドの重厚感を伝えるため、縦書きのレイアウトと静寂な余白を意識。実績のディテールを大きく魅せるためにフォトビューアーを連動。



【06. 医療・福祉】
クライアント名： ソラノ診療所（または 赤坂テラスクリニック）

カテゴリタグ： 施設サイト UIデザイン アクセシビリティ
UI / UX Design / Accessibility

実績タイトル： 「欲しい情報に3秒で辿り着く」を形にした、地域に寄り添うクリニックのサイト設計

プロジェクト概要： 不安を抱える患者さんや親御さんが迷わないことを最優先に。診療時間や担当医のテーブル（表）を徹底的に見やすく整理し、よくある質問にはアコーディオンを実装。



07. 【アパレル・インテリア】
クライアント名： UNWIND（ライフスタイルブランド）

カテゴリタグ： ブランドサイト スライドショー Webデザイン
EC Website / Visual Experience


実績タイトル： 空間の空気感をそのままWebへ落とし込んだ、エキシビション型ブランドサイト

プロジェクト概要： 家具やテキスタイルの持つ独特の世界観を伝えるため、トップページには洗練された大判のスライドショーを配置。無駄な装飾を削ぎ落としたスクエアルックなグリッド（格子状）レイアウトで、プロダクトの造形美を静かに際立たせる設計に。




08. 【専門職・士業】
クライアント名： 赤坂リーガルパートナーズ（総合法律事務所）

カテゴリタグ： コーポレートサイト 信頼感 コーディング
Corporate Website / Trust & Reliability

実績タイトル： 堅苦しさを排除し、圧倒的な誠実さとクリアな知性を伝える士業サイト

プロジェクト概要： モノトーンをベースに、1pxのシャープな線でエリアを区切った硬質なデザイン。所属弁護士のプロフィールを画面遷移なしのモーダルでスマートに表示。




09. 【フード・飲料】
クライアント名： GROUNDS（サスティナブルコーヒーロースター）

Brand & Promotion / Culture Marketing
カテゴリタグ： ブランドサイト オウンドメディア コンポーネント意識

実績タイトル： 豆のストーリーを丁寧に読ませる、グリッド構造のブランドメディア

プロジェクト概要： 産地や農園の紹介を、コンポーネントのルールを徹底した統一感のあるレイアウトで配置。写真はすべて正方形（1:1）に統一し、美しく整列。




10. 【教育・スクール】
クライアント名： BASE ACADEMY（通信制Webデザインスクール）

カテゴリタグ： ランディングページ UI/UXデザイン CVR改善
UI / UX Design / Conversion LP

実績タイトル： 受講検討者の不安をロジカルに解消する、直感的なサービスサイト設計

プロジェクト概要： カリキュラムの詳細や受講生の声を、折りたたみ式のアコーディオンでスッキリ整理。問い合わせ（Contact）への導線を迷わせない直線的な動線で設計。
